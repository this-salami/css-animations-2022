# css-animations-2022

Collections of CSS animations I made in sophomore year of highschool (fall of '22)

While I had UI/UX experience for video games, front-end was a little new for me, so this was one of my first experiments with HTML/JS/CSS.

## Loading Animations

A collection of animations used to indicate a web page is loading, either took inspiration from different sites I used/saw at the time and my own creations. 

GitHub Pages: [Loading Animations](https://this-salami.github.io/css-animations-2022/CssLoadingAnims/).

## EXP Animation

Inspired by the exp bar in many video games, I thought it would look cool as a loading/progress bar!

GitHub Pages: [EXP Animation](https://this-salami.github.io/css-animations-2022/ExpAnimation/).
